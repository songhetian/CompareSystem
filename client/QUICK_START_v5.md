# 🚀 快速开始指南 - v5.0 现代化版本

## 安装与启动

### 1. 安装依赖
```bash
cd client
npm install
```

### 2. 开发模式启动
```bash
npm run dev
```
应用将在 Electron 窗口中打开。

### 3. 生产构建
```bash
npm run build
```

## 🎯 功能导览

### 🧠 智能人力测算 (BudgetPage)

**用途**: 根据业务目标计算所需人力编制

**操作流程**:
1. **步骤1 - 目标设定**
   - 选择测算模板 (可选)
   - 选择驱动方式: 💰 销售额 或 👥 访客数
   - 输入目标数值
   - 选择活动力度 (可选)

2. **步骤2 - 时间规划**
   - 选择活动周期 (开始-结束日期)
   - 添加流量高峰日期 (可多个)

3. **步骤3 - 班次配置**
   - 勾选需要的工作班次

4. **步骤4 - 查看结果**
   - 查看建议总编制
   - 查看岗位分布
   - 查看多日趋势图
   - 查看24小时分布图
   - 导出报表 (开发中)

**快捷操作**:
- 点击 "上一步" 返回修改
- 每步自动验证，避免错误
- 进度条实时显示完成度

---

### ⚙️ 参数配置中心 (ParamPage)

**用途**: 管理和编辑测算参数模板

**操作流程**:
1. **左侧**: 选择或创建参数方案
   - 点击方案卡片查看详情
   - ⭐ 设为默认方案
   - 🗑️ 删除方案 (默认方案不可删)
   - ➕ 新建方案

2. **右侧**: 编辑参数配置
   - 6大分类: 💼业务 🎯漏斗 ⚡效能 🔥爆发 🎉大促 ⏰偏移
   - 拖动滑块或直接输入数值
   - 修改后显示 "未保存" 标签
   - 点击 "保存修改" 提交更改
   - 点击 "重置" 恢复原值

**参数说明**:
- 💼 **基础业务**: 客单价、访客数、峰值系数等
- 🎯 **转化漏斗**: 访客→咨询→下单→付款→售后
- ⚡ **岗位效能**: 处理时长、饱和度
- 🔥 **分时爆发**: 各岗位爆发时段和倍数
- 🎉 **大促系数**: S级/A级活动倍数
- ⏰ **时间偏移**: 售前/售中/售后时间偏移

**技巧**:
- 百分比参数用 0-1 表示 (0.5 = 50%)
- 鼠标悬停参数名称查看说明
- 新建方案会复制当前方案参数

### ⏰ 排班室 (ShiftPage)

**用途**: 管理工作班次配置

**操作流程**:
1. 查看现有班次
   - 顶部显示4类班次统计 (早/中/晚/全天)
   - 卡片展示每个班次详情

2. 创建新班次
   - 点击 "创建班次" 按钮
   - 填写班次信息:
     - 📝 班次名称 (如: 早班A组)
     - 🏷️ 班次类型 (早/中/晚/全天)
     - ⏰ 工作时间段
     - ⏱️ 工作时长 (小时)
     - 💤 休息时长 (小时, 可选)
   - 确认创建

3. 管理班次
   - 编辑班次 (开发中)
   - 删除班次

**建议**:
- 🌅 早班: 7:00-15:00 (覆盖早高峰)
- ☀️ 中班: 10:00-18:00 (午间主力)
- 🌙 晚班: 14:00-22:00 (晚间售后)
- 🌍 全天: 9:00-18:00 (标准工作日)

### 🔥 活动规划 (PromotionPage)

**用途**: 配置营销活动和流量系数

**操作流程**:
1. 查看活动级别说明
   - 💥 S级: 2.5x+ (双11、618超级大促)
   - 🚀 A级: 1.8-2.5x (会员日、品牌日)
   - 🎪 B级: 1.3-1.8x (品类日促销)
   - 📦 C级: 1.0-1.3x (日常运营)

2. 创建活动方案
   - 点击 "创建活动" 按钮
   - 填写活动信息:
     - 📝 活动名称 (如: 双11超级大促)
     - 🔥 流量爆发系数 (拖动滑块 1.0-5.0)
     - 📄 方案说明 (可选, 最多200字)
   - 系统自动判定活动级别
   - 确认创建

3. 管理活动
   - 查看活动详情卡片
   - 编辑活动 (开发中)
   - 删除活动

**技巧**:
- 系数越高，预估人力需求越大
- 实时显示级别和倍数
- 可根据历史数据调整系数

---

### 📊 历史记录 (ReportPage)

**用途**: 查看和分析过往测算记录

**操作流程**:
1. 查看统计概览
   - 📊 累计记录数
   - 📅 最近测算时间
   - 📈 本月记录数

2. 浏览时间线
   - 垂直时间线展示所有记录
   - 🟢 绿色圆点: 最近7天记录
   - ⚪ 灰色圆点: 更早记录

3. 查看详情
   - 点击 "查看详情" 按钮
   - 查看基本信息
   - 查看参数配置 (JSON格式)
   - 查看测算结果 (JSON格式)

4. 删除记录
   - 点击 "删除" 按钮
   - 确认删除操作

**建议**:
- 定期清理过期记录
- 对比不同时期配置
- 导出数据用于分析 (开发中)

## 💡 使用技巧

### 基本工作流
```
1. 配置班次 (ShiftPage)
   ↓
2. 配置活动 (PromotionPage) [可选]
   ↓
3. 配置参数 (ParamPage) [可选]
   ↓
4. 开始测算 (BudgetPage)
   ↓
5. 查看结果和图表
   ↓
6. 历史记录存档 (自动)
```

### 快捷键 (计划中)
- `Ctrl + N`: 新建方案
- `Ctrl + S`: 保存修改
- `Ctrl + R`: 刷新数据
- `Esc`: 关闭模态框

### 常见问题

**Q: 测算结果为0或异常？**
A: 检查目标数值是否正确，确保选择了至少一个班次

**Q: 参数修改后没生效？**
A: 记得点击 "保存修改" 按钮，看到 "保存成功" 提示

**Q: 找不到历史记录？**
A: 只有完成测算才会保存记录，检查是否到达步骤4

**Q: 图表显示不正确？**
A: 确保测算数据完整，刷新页面重试

**Q: 删除误操作如何恢复？**
A: 目前不支持撤销，请谨慎操作，未来会添加回收站功能

## 🔧 故障排除

### 应用无法启动
```bash
# 1. 清理依赖重新安装
rm -rf node_modules package-lock.json
npm install

# 2. 清理构建产物
rm -rf out

# 3. 重启开发服务器
npm run dev
```

### 数据加载失败
1. 检查数据库文件是否存在 (`data/service_budget.db`)
2. 查看 Electron 控制台错误信息 (F12)
3. 尝试重启应用

### 页面显示异常
1. 按 F5 刷新页面
2. 清除浏览器缓存
3. 检查网络连接
4. 查看控制台错误

### 测算结果异常
1. 检查输入参数是否合理
2. 查看参数配置是否正确
3. 检查班次是否已选择
4. 查看控制台计算日志
