# V7.0 Arco Design 全面重构方案

## 📋 项目信息
- **版本号**: V7.0.0
- **设计日期**: 2026-06-03
- **重构范围**: 全站UI组件和布局
- **设计系统**: Arco Design
- **优先级**: ⭐⭐⭐⭐ 非常高

---

## 🎯 重构目标

### 核心目标
1. **统一设计语言** - 使用 Arco Design 建立一致的视觉风格
2. **提升用户体验** - 现代化、流畅、响应式的界面
3. **提高开发效率** - 组件化、可复用、易维护
4. **优化性能** - 减少不必要的渲染，提升交互流畅度

### 设计原则
- **简洁** - 清晰的信息架构，减少视觉干扰
- **一致** - 统一的间距、颜色、字体、圆角
- **高效** - 减少点击次数，优化操作流程
- **现代** - 使用最新的设计趋势和交互模式

## 🏗️ 布局重构

### 主布局结构

```
┌──────────────────────────────────────────┐
│  Sider (侧边栏)        │  Content Area   │
│  ├─ Logo              │  ├─ Header       │
│  ├─ Navigation        │  ├─ PageContent  │
│  └─ Footer/Settings   │  └─ Footer       │
└──────────────────────────────────────────┘
```

### Sider（侧边栏）设计

**特点**:
- 深色主题或浅色主题
- 可折叠
- Logo + 导航菜单
- 底部设置区

**规范**:
- 宽度: 240px (展开) / 64px (折叠)
- 背景色: `--gray-2` 或 `#1D2129`
- 菜单项高度: 40px
- 图标大小: 18px
- 间距: 8px

### Header（顶部栏）设计

**特点**:
- 页面标题
- 面包屑导航
- 快捷操作
- 用户信息

**规范**:
- 高度: 64px
- 背景色: `#FFFFFF`
- 边框: `1px solid --gray-3`
- 内边距: `0 24px`

### Content（内容区）设计

**特点**:
- 页面主体内容
- 卡片式布局
- 响应式设计

**规范**:
- 背景色: `--gray-1`
- 内边距: `24px`
- 卡片间距: `16px`

## 🧩 组件重构清单

### 1. 按钮组件 (Button)

**类型**:
- Primary（主要按钮）
- Secondary（次要按钮）
- Outline（描边按钮）
- Text（文本按钮）
- Dashed（虚线按钮）

**尺寸**:
- Mini (24px)
- Small (28px)
- Default (32px)
- Large (36px)

**状态**:
- Default（默认）
- Hover（悬停）
- Active（激活）
- Disabled（禁用）
- Loading（加载中）

**使用规范**:
```tsx
// 主要操作
<Button type='primary' size='default' icon={<IconPlus />}>
  新建项目
</Button>

// 次要操作
<Button type='secondary'>
  取消
</Button>

// 危险操作
<Button type='primary' status='danger'>
  删除
</Button>

// 成功操作
<Button type='primary' status='success'>
  保存
</Button>
```

### 2. 表格组件 (Table)

**特点**:
- 斑马纹
- 悬停高亮
- 固定表头
- 列宽可调
- 排序功能
- 筛选功能
- 分页器

**样式规范**:
```tsx
<Table
  stripe
  hover
  borderCell
  pagination={{
    pageSize: 20,
    showTotal: true,
    showJumper: true,
    sizeCanChange: true,
    pageSizeOptions: [10, 20, 50, 100]
  }}
  scroll={{ x: 1200, y: 600 }}
/>
```

**表头样式**:
- 背景色: `--gray-2`
- 字重: 500
- 高度: 48px
- 边框: `1px solid --gray-3`

**单元格样式**:
- 高度: 48px
- 内边距: `12px 16px`
- 对齐: 左对齐（默认）

### 3. 表单组件 (Form)

**布局模式**:
- Horizontal（水平）
- Vertical（垂直）
- Inline（行内）

**标签规范**:
- 宽度: 100px (水平模式)
- 对齐: 右对齐
- 必填标记: 红色星号

**样式规范**:
```tsx
<Form
  layout='vertical'
  size='large'
  labelCol={{ span: 4 }}
  wrapperCol={{ span: 20 }}
  autoComplete='off'
>
  <Form.Item
    label='项目名称'
    field='name'
    rules={[{ required: true, message: '请输入项目名称' }]}
  >
    <Input placeholder='请输入项目名称' />
  </Form.Item>
</Form>
```

### 4. 输入框组件 (Input)

**类型**:
- Input（单行输入）
- Input.TextArea（多行输入）
- Input.Search（搜索框）
- Input.Password（密码框）
- InputNumber（数字输入）

**尺寸**:
- Small (28px)
- Default (32px)
- Large (36px)

**状态**:
- Normal（正常）
- Focus（聚焦）
- Disabled（禁用）
- Error（错误）

**样式规范**:
```tsx
// 基础输入
<Input
  placeholder='请输入内容'
  prefix={<IconSearch />}
  allowClear
/>

// 数字输入
<InputNumber
  placeholder='请输入数字'
  min={0}
  max={100}
  step={1}
  precision={2}
/>

// 多行输入
<Input.TextArea
  placeholder='请输入内容'
  rows={4}
  maxLength={200}
  showWordLimit
/>
```

### 5. 下拉选择 (Select)

**模式**:
- Single（单选）
- Multiple（多选）
- Tags（标签模式）

**功能**:
- 搜索过滤
- 远程搜索
- 虚拟滚动
- 分组显示

**样式规范**:
```tsx
<Select
  placeholder='请选择'
  showSearch
  allowClear
  filterOption={(inputValue, option) =>
    option.props.children.toLowerCase().indexOf(inputValue.toLowerCase()) >= 0
  }
>
  <Select.Option value='1'>选项1</Select.Option>
  <Select.Option value='2'>选项2</Select.Option>
</Select>
```

### 6. 日期选择器 (DatePicker)

**类型**:
- DatePicker（日期）
- DatePicker.RangePicker（日期范围）
- DatePicker.WeekPicker（周选择器）
- DatePicker.MonthPicker（月选择器）
- DatePicker.YearPicker（年选择器）

**样式规范**:
```tsx
<DatePicker
  placeholder='请选择日期'
  format='YYYY-MM-DD'
  allowClear
/>

<DatePicker.RangePicker
  placeholder={['开始日期', '结束日期']}
  format='YYYY-MM-DD'
  allowClear
/>
```

### 7. 卡片组件 (Card)

**类型**:
- Basic（基础卡片）
- Bordered（带边框）
- Hoverable（可悬停）

**样式规范**:
```tsx
<Card
  bordered={false}
  hoverable
  style={{
    borderRadius: 'var(--radius-large)',
    boxShadow: 'var(--shadow-2)'
  }}
  title={<Text bold>卡片标题</Text>}
  extra={<Button type='text'>更多</Button>}
>
  卡片内容
</Card>
```

### 8. 模态框 (Modal)

**尺寸**:
- Small (400px)
- Default (520px)
- Large (800px)
- Fullscreen (全屏)

**样式规范**:
```tsx
<Modal
  title='标题'
  visible={visible}
  onOk={handleOk}
  onCancel={handleCancel}
  style={{ width: 600 }}
  okText='确定'
  cancelText='取消'
>
  模态框内容
</Modal>
```

### 9. 消息提示 (Message)

**类型**:
- Info（信息）
- Success（成功）
- Warning（警告）
- Error（错误）

**使用规范**:
```tsx
Message.success('操作成功！');
Message.error('操作失败，请重试');
Message.warning('请注意检查数据');
Message.info('提示信息');
```

### 10. 统计数值 (Statistic)

**样式规范**:
```tsx
<Statistic
  title='总销售额'
  value={12580}
  precision={2}
  suffix='万'
  prefix={<IconUp />}
  styleValue={{ color: '#00B42A', fontSize: 28 }}
/>
```
