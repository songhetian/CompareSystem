## 📋 具体重构步骤

### 通用重构模式

每个页面重构遵循以下步骤：

#### Step 1: 页面结构调整
```tsx
// 使用统一的页面容器
<div className='page-container'>
  {/* 页面头部 */}
  <PageHeader
    title='页面标题'
    subtitle='页面副标题'
    icon={<IconDashboard />}
    extra={<Button>操作按钮</Button>}
  />

  {/* 页面内容 */}
  <div className='page-content'>
    {/* 具体内容 */}
  </div>
</div>
```

#### Step 2: 卡片式布局
```tsx
<Card
  bordered={false}
  style={{
    borderRadius: 'var(--radius-large)',
    boxShadow: 'var(--shadow-medium)',
    marginBottom: 'var(--spacing-medium)',
  }}
  title='卡片标题'
>
  {/* 卡片内容 */}
</Card>
```

#### Step 3: 表单优化
```tsx
<Form
  layout='vertical'
  size='large'
  autoComplete='off'
>
  <Form.Item
    label='标签'
    field='fieldName'
    rules={[{ required: true }]}
  >
    <Input placeholder='请输入' allowClear />
  </Form.Item>
</Form>
```

#### Step 4: 按钮组优化
```tsx
<Space size={12}>
  <Button type='primary' icon={<IconPlus />} size='large'>
    新建
  </Button>
  <Button icon={<IconDownload />} size='large'>
    导出
  </Button>
</Space>
```

#### Step 5: 表格优化
```tsx
<Table
  columns={columns}
  data={data}
  stripe
  hover
  borderCell
  pagination={{
    pageSize: 20,
    showTotal: true,
    showJumper: true,
    sizeCanChange: true,
  }}
  scroll={{ x: 1200 }}
/>
```

## 🎨 样式规范提醒

### 间距使用

```tsx
// ❌ 错误：硬编码数值
<div style={{ margin: '16px' }}>

// ✅ 正确：使用设计令牌
<div style={{ margin: 'var(--spacing-medium)' }}>
```

### 颜色使用

```tsx
// ❌ 错误：硬编码颜色
<Button style={{ background: '#1890ff' }}>

// ✅ 正确：使用 Arco 的 type 属性
<Button type='primary'>
```

### 圆角使用

```tsx
// ❌ 错误：硬编码圆角
<Card style={{ borderRadius: '8px' }}>

// ✅ 正确：使用设计令牌
<Card style={{ borderRadius: 'var(--radius-large)' }}>
```

## 🔧 重构工具和技巧

### 1. 快速查找替换

**查找旧的间距**:
```
查找: style={{.*margin.*:.*'[\d]+px'
替换: 使用 var(--spacing-*)
```

**查找内联样式**:
```
查找: style={{
检查: 是否可以用 Arco 组件属性替代
```

### 2. 组件属性对照表

| 旧写法 | 新写法 |
|--------|--------|
| `className='p-4'` | `style={{ padding: 'var(--spacing-medium)' }}` |
| `className='rounded-lg'` | `style={{ borderRadius: 'var(--radius-large)' }}` |
| `className='shadow-md'` | `style={{ boxShadow: 'var(--shadow-medium)' }}` |

### 3. Arco 组件推荐用法

**Button**:
```tsx
<Button type='primary' size='large' status='success'>
  确定
</Button>
```

**Input**:
```tsx
<Input
  placeholder='请输入'
  prefix={<IconSearch />}
  allowClear
  size='large'
/>
```

**Select**:
```tsx
<Select
  placeholder='请选择'
  showSearch
  allowClear
  filterOption={(inputValue, option) =>
    option.props.children.toLowerCase().includes(inputValue.toLowerCase())
  }
>
  <Select.Option value='1'>选项1</Select.Option>
</Select>
```

## 💡 重构建议

### Do（推荐）

✅ **使用设计令牌**
```tsx
style={{ padding: 'var(--spacing-medium)' }}
```

✅ **使用 Arco 组件属性**
```tsx
<Button type='primary' size='large' />
```

✅ **统一卡片样式**
```tsx
<Card bordered={false} hoverable />
```

✅ **使用 Space 组件管理间距**
```tsx
<Space size={16} direction='vertical' />
```

### Don't（避免）

❌ **硬编码数值**
```tsx
style={{ margin: '16px' }}
```

❌ **使用 TailwindCSS 类名**
```tsx
className='p-4 mt-2'
```

❌ **混用多种样式方案**
```tsx
// 同时使用 inline style, className, sx 等
```

❌ **过度嵌套组件**
```tsx
// 保持组件层级清晰
```
