# 班次表单优化说明

## 🎯 优化内容

### 问题描述
- ❌ 表单布局因"*必填"标记错位
- ❌ 时间输入不方便（手动输入容易出错）
- ❌ 工作时长需要手动计算和输入

### 解决方案

#### 1. 布局优化 ✅

**优化前**：
- 使用 Tailwind 的 `grid grid-cols-2 gap-4` 类
- "*必填"标记导致列宽不一致

**优化后**：
- 改用内联样式 `display: grid; gridTemplateColumns: '1fr 1fr'`
- 设置 `requiredSymbol={{ position: 'start' }}` 确保标记在前
- 明确设置 label 样式：`fontSize: 14, fontWeight: 500`
- 调整模态框宽度：650px → 550px

**效果**：
- ✅ 列宽完全一致
- ✅ 必填标记统一位置
- ✅ 视觉对齐完美

---

#### 2. 时间选择器 ✅

**优化前**：
```typescript
<Input placeholder="例如: 08:00" />
```
- 需要手动输入时间
- 容易输入错误格式
- 用户体验不佳

**优化后**：
```typescript
<TimePicker
  format="HH:mm"
  placeholder="选择开始时间"
  onChange={handleTimeChange}
/>
```

**优势**：
- ✅ 可视化选择时间
- ✅ 格式统一（HH:mm）
- ✅ 避免输入错误
- ✅ 用户体验更好

---

#### 3. 自动计算工作时长 ✅

**新增功能**：
```typescript
// 计算工作时长
const calculateWorkHours = (startTime: string, endTime: string): number => {
  if (!startTime || !endTime) return 0;

  const start = dayjs(`2000-01-01 ${startTime}`);
  const end = dayjs(`2000-01-01 ${endTime}`);

  let diff = end.diff(start, 'hour', true);
  if (diff < 0) {
    // 跨天情况（如 22:00 - 06:00）
    diff = 24 + diff;
  }

  return Math.round(diff * 10) / 10; // 保留1位小数
};

// 监听时间变化
const handleTimeChange = () => {
  const startTime = form.getFieldValue('startTime');
  const endTime = form.getFieldValue('endTime');

  if (startTime && endTime) {
    const hours = calculateWorkHours(startTime, endTime);
    form.setFieldValue('hours', hours);
  }
};
```

**工作时长输入框设置**：
```typescript
<InputNumber
  placeholder="自动计算"
  disabled                          // 禁用手动编辑
  style={{ background: '#f5f5f5' }} // 灰色背景表示禁用
/>
```

**计算规则**：
- 自动计算两个时间的差值
- 支持跨天计算（如晚班 22:00 → 次日 06:00 = 8小时）
- 保留1位小数（如 8.5小时）
- 实时更新，无需点击按钮

**示例**：
- 08:00 → 17:00 = **9.0小时**
- 10:00 → 18:00 = **8.0小时**
- 22:00 → 06:00 = **8.0小时**（跨天）
- 14:00 → 22:30 = **8.5小时**

## 📝 使用流程

### 创建班次（新流程）

1. **输入班次名称**
   - 例如：早班A组
   - 必填项

2. **选择开始时间**
   - 点击时间选择器
   - 选择时间（如 08:00）
   - 自动触发工作时长计算

3. **选择结束时间**
   - 点击时间选择器
   - 选择时间（如 17:00）
   - 自动触发工作时长计算
   - **工作时长自动显示**（如 9.0小时）

4. **设置休息时长**（可选）
   - 输入休息时长（如 1小时）
   - 非必填

5. **确认创建**
   - 点击"创建"按钮
   - 系统自动判断班次类型
   - 保存到数据库
