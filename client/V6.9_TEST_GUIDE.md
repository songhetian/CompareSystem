### 测试用例 2: 同比分析 - 按月

**测试步骤**:
1. 点击"📈 同比/环比"按钮
2. 选择"同比 (YoY)"
3. 选择"按月"
4. 观察图表和表格

**预期结果**:
- ✅ 图表显示三条曲线（销售额/人数/效率增长率）
- ✅ 表格显示各月份的同比数据
- ✅ 只显示有去年同期数据的月份
- ✅ 增长率计算正确：`(2026-01 - 2025-01) / 2025-01 * 100`

**验证示例**:
```
假设数据：
2025-01: 销售额 1000万, 人数 40人, 效率 25万/人
2026-01: 销售额 1200万, 人数 42人, 效率 28.57万/人

预期计算：
销售额增长率 = (1200-1000)/1000*100 = +20.00%
人数增长率 = (42-40)/40*100 = +5.00%
效率增长率 = (28.57-25)/25*100 = +14.28%
```

## 🐛 已知问题和限制

### 数据要求
- 同比分析需要至少2年的数据
- 环比分析需要至少2个周期的数据
- 人数为0的数据不计入效率计算

### 功能限制
- 暂不支持季度环比（QoQ）
- 暂不支持自定义周期
- 暂不支持导出分析结果

### 测试技巧

**快速计算增长率验证**:
```javascript
// 在浏览器控制台中验证
const current = 1200;  // 当前值
const previous = 1000; // 对比值
const growth = ((current - previous) / previous * 100).toFixed(2);
console.log(`增长率: ${growth}%`);
```

**检查数据正确性**:
```javascript
// 在浏览器控制台查看原始数据

  currentSales: 1200,
  previousSales: 1000,
  expectedGrowth: 20.00
```
