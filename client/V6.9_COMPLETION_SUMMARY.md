## 🔍 核心算法说明

### 同比计算流程

```
1. 数据分组
   └─ 按月: 2026-01, 2026-02...
   └─ 按周: 2026-W01, 2026-W02...

2. 计算周期平均值
   └─ 平均销售额 = Σ销售额 / 记录数
   └─ 平均人数 = Σ人数 / 记录数
   └─ 平均效率 = 平均销售额 / 平均人数

3. 查找去年同期
   └─ 2026-01 → 查找 2025-01
   └─ 2026-W10 → 查找 2025-W10

4. 计算增长率
   └─ 增长率 = (当前值 - 去年值) / 去年值 × 100%
```

### 环比计算流程

```
1. 数据分组并排序
   └─ 按时间从早到晚排序

2. 计算周期平均值
   └─ 同上

3. 与上一周期对比
   └─ 2026-02 → 对比 2026-01
   └─ 2026-W11 → 对比 2026-W10

4. 计算增长率
   └─ 增长率 = (当前值 - 上期值) / 上期值 × 100%
```

## 💡 智能洞察算法

### 洞察维度

1. **销售额趋势**
   ```typescript
   if (avgSalesGrowth > 10%) → "强劲增长" ✨
   else if (avgSalesGrowth > 0%) → "平稳增长" ✓
   else → "出现下滑" ⚠️
   ```

2. **效率分析**
   ```typescript
   if (avgEfficiencyGrowth > 5%) → "持续提升" 🎯
   else if (avgEfficiencyGrowth > 0%) → "稳步提升" ✓
   else if (avgEfficiencyGrowth > -5%) → "略有下降" ⚠️
   else → "明显下降" ❗
   ```

3. **增长类型**
   ```typescript
   if (salesGrowth > staffGrowth + 5%) → "效率驱动型" ⭐
   else if (staffGrowth > salesGrowth + 5%) → "人员冗余风险" ⚠️
   ```

4. **稳定性**
   ```typescript
   std = 增长率标准差
   if (std < 5%) → "波动较小，稳定" ✓
   else if (std > 15%) → "波动较大" ℹ️
   ```
