### 1. 置信度评分系统 📊

#### 什么是置信度？
置信度是对预测结果可靠性的量化评估，范围 0-100%。

```
置信度 = 100% - 扣分因素

扣分因素：
• 数据量不足
• 数据波动性大
• 数据完整性差
```

#### 置信度计算逻辑

```typescript
let confidence = 100;

// 1. 数据量因素
if (数据量 < 30条)  confidence -= 20;
if (数据量 < 10条)  confidence -= 30;

// 2. 波动性因素（变异系数）
变异系数 = 标准差 / 平均值
if (变异系数 > 0.3)  confidence -= 15;
if (变异系数 > 0.5)  confidence -= 25;

// 3. 完整性因素
完整性 = 有效记录数 / 总记录数
if (完整性 < 0.8)  confidence -= 10;
if (完整性 < 0.6)  confidence -= 20;

// 4. 最低保障
confidence = Math.max(confidence, 30);
```

#### 置信度等级

| 置信度 | 等级 | 说明 | 颜色 | 图标 |
|--------|------|------|------|------|
| 80-100% | 高 | 预测结果可靠，可直接使用 | 🟢 绿色 | 🎯 |
| 60-79% | 中 | 建议结合业务判断 | 🟡 黄色 | ⚠️ |
| 30-59% | 低 | 建议谨慎参考 | 🔴 红色 | ❗ |

### 3. 异常数据标记 🚨

#### 什么是异常数据？
偏离历史平均值超过 **2.5个标准差** 的数据点。

#### 异常检测算法

```typescript
// 1. 计算平均效率
平均效率 = Σ(销售额/人数) / N

// 2. 计算标准差
标准差 = √(Σ(效率 - 平均效率)² / N)

// 3. 计算Z分数
Z分数 = |效率 - 平均效率| / 标准差

// 4. 判断异常
if (Z分数 > 2.5) {
  标记为异常
}
```

#### 统计学依据

```
正态分布规律：
• ±1σ: 包含 68.27% 数据
• ±2σ: 包含 95.45% 数据
• ±2.5σ: 包含 98.76% 数据

超过 2.5σ 的数据点：
• 仅占 1.24%
• 极大可能是异常值
```

#### 异常原因分析

**可能的异常原因：**
- 🎉 **特殊活动** - 大促、周年庆等
- 📦 **库存异常** - 缺货或爆款
- 🔧 **系统故障** - 技术问题影响
- 📊 **数据录入错误** - 人为失误
- 🌐 **外部因素** - 节假日、天气等

#### 界面标记

**表格中的异常标记：**
```
人均效率列：
正常数据: 29.39万/人
异常数据: [28.55万/人 ⚠️]（橙色Tag）
         悬停提示：异常数据：偏离平均值超过2.5个标准差
```

**统计中的异常提示：**
```
⚠️ 检测到 3 条异常数据（偏离平均值超过2.5个标准差）
```

**预测结果中的异常警告：**
```
[置信度: 85%] [⚠️ 3条异常]
```

#### 处理建议

**发现异常数据时：**

1. **查看备注**
   ```
   点击异常数据行
     ↓
   查看"备注"字段
     ↓
   了解异常原因
   ```

2. **判断影响**
   ```
   偶发异常（1-2条）
     → 影响较小，可忽略

   系统性异常（>5条）
     → 影响较大，需处理
   ```

3. **处理方式**
   ```
   方式1: 删除异常数据
   方式2: 在备注中标注原因，保留数据
   方式3: 单独创建"特殊活动"项目
   ```

## 🔬 技术实现细节

### 1. 置信度计算函数

```typescript
const calculateHistoryStats = (data: HistoryDataRecord[]) => {
  // ... 基础统计计算

  // 计算变异系数
  const efficiencies = data
    .filter(d => d.actual_staff > 0)
    .map(d => d.sales_volume / d.actual_staff);

  const variance = efficiencies.reduce(
    (sum, eff) => sum + Math.pow(eff - avgEfficiency, 2),
    0
  ) / efficiencies.length;

  const stdDev = Math.sqrt(variance);
  const coefficientOfVariation = stdDev / avgEfficiency;

  // 计算置信度
  let confidence = 100;
  if (data.length < 30) confidence -= 20;
  if (data.length < 10) confidence -= 30;
  if (coefficientOfVariation > 0.3) confidence -= 15;
  if (coefficientOfVariation > 0.5) confidence -= 25;

  const completeness = data.filter(d => d.actual_staff > 0).length / data.length;
  if (completeness < 0.8) confidence -= 10;
  if (completeness < 0.6) confidence -= 20;

  confidence = Math.max(confidence, 30);

  return { ..., confidence, stdDev };
};
```

### 2. 异常检测函数

```typescript
// 检测异常数据
const anomalies = data.filter(d => {
  if (d.actual_staff === 0) return false;

  const efficiency = d.sales_volume / d.actual_staff;
  const zScore = Math.abs((efficiency - avgEfficiency) / stdDev);

  return zScore > 2.5; // Z分数阈值
});
```
