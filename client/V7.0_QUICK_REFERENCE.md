## 🔧 组件使用

### PageHeader 组件

```tsx
import { PageHeader } from '../components/common';

<PageHeader
  title="页面标题"
  subtitle="页面描述"
  icon="🎯"
  breadcrumb={[
    { title: '首页' },
    { title: '当前页' }
  ]}
  extra={
    <Space>
      <Button>刷新</Button>
      <Button type='primary'>新建</Button>
    </Space>
  }
/>
```

**属性**:
- `title` - 页面标题（必填）
- `subtitle` - 副标题（可选）
- `icon` - 图标 Emoji（可选）
- `breadcrumb` - 面包屑导航（可选）
- `extra` - 右侧操作区（可选）

### StatsCard 组件

```tsx
import { StatsCard } from '../components/common';

<StatsCard
  title="总销售额"
  value={1000}
  suffix="万"
  icon="💰"
  color="#165DFF"
  trend="up"
  trendValue="+15.3%"
  bgGradient="linear-gradient(135deg, #667eea 0%, #764ba2 100%)"
/>
```

**属性**:
- `title` - 标题（必填）
- `value` - 数值（必填）
- `suffix` - 后缀单位（可选）
- `prefix` - 前缀（可选）
- `icon` - 图标 Emoji（可选）
- `color` - 主题色（可选，默认 #165DFF）
- `trend` - 趋势：'up' | 'down' | 'neutral'（可选）
- `trendValue` - 趋势值如 "+15.3%"（可选）
- `bgGradient` - 背景渐变（可选）

## ✨ 样式最佳实践

### ✅ 推荐做法

```tsx
// 1. 使用 CSS 类
<div className='page-container'>
  <div className='data-card'>
    <div className='empty-state'>暂无数据</div>
  </div>
</div>

// 2. 使用设计令牌
<div style={{
  padding: 'var(--spacing-large)',
  borderRadius: 'var(--radius-large)',
  boxShadow: 'var(--shadow-medium)',
  background: 'var(--gray-1)'
}}>

// 3. 使用 Tailwind 工具类
<div className='grid grid-cols-4 gap-4'>
<div className='flex items-center justify-between'>
<div className='mb-4'>  // margin-bottom: 1rem
```

### ❌ 避免做法

```tsx
// 1. 避免硬编码样式
<div style={{ padding: '24px', borderRadius: '8px' }}>

// 2. 避免自定义 className
<div className='my-custom-header'>

// 3. 避免混乱的内联样式
<div style={{
  background: '#f8f9fa',
  padding: '24px',
  marginBottom: '16px',
  borderRadius: '12px'
}}>
```

## 💡 常见问题

### Q: 如何添加新页面？
A: 复制上面的"标准页面布局"模板，替换内容即可。

### Q: 如何自定义颜色？
A: 修改 `src/theme/design-tokens.ts` 中的颜色定义。

### Q: 如何添加新的 CSS 类？
A: 在 `src/theme/global.css` 中添加，使用设计令牌。

### Q: 如何统一修改所有页面的头部样式？
A: 修改 `PageHeader.tsx` 组件即可全局生效。

### Q: 发现 Bug 怎么办？
A: 检查浏览器控制台错误，查看编译错误，阅读相关文档。
