## ✨ 新增功能

### 1. 置信度评分系统 📊

**功能说明**：自动评估预测结果的可靠性，范围 0-100%

**评分因素**：
- ✅ 数据量充足度（30条以上最佳）
- ✅ 数据波动性（变异系数）
- ✅ 数据完整性（有效记录比例）

**置信度等级**：
```
80-100% 🎯 高置信度 - 预测结果可靠
60-79%  ⚠️  中置信度 - 建议结合业务判断
30-59%  ❗ 低置信度 - 建议谨慎参考
```

**界面展示**：
- 预测弹窗中醒目显示置信度
- 渐变卡片设计（紫蓝色）
- 置信度描述和图标
- 质量提升建议（低置信度时）

---

### 2. 趋势图可视化 📈

**功能说明**：可视化展示历史业务数据趋势

**三条曲线**：
- 🔵 **销售额** - 观察业务增长
- 🟠 **人数** - 观察人力配置
- 🟢 **人均效率** - 观察效率变化

**图表特性**：
- ✅ 平滑曲线展示
- ✅ 三Y轴独立缩放
- ✅ 鼠标悬停显示数据
- ✅ 图例交互控制
- ✅ 渐变填充效果
- ✅ 响应式自适应

**使用方式**：
```
历史数据弹窗
  ↓
点击"查看趋势图分析"
  ↓
图表展开显示
  ↓
分析业务规律
```

---

### 3. 异常数据标记 🚨

**功能说明**：自动识别偏离平均值超过2.5个标准差的异常数据

**检测算法**：
```
Z分数 = |效率 - 平均效率| / 标准差

if (Z分数 > 2.5) {
  标记为异常
}
```

**界面标记**：
- ⚠️ 表格中橙色Tag标注
- 悬停显示异常提示
- 统计中显示异常数量
- 预测卡片中显示异常警告

**异常原因参考**：
- 特殊活动（大促、周年庆）
- 库存异常（缺货、爆款）
- 系统故障
- 数据录入错误
- 外部因素（节假日）

## 🔬 技术亮点

### 1. 统计学算法

**变异系数（Coefficient of Variation）**：
```typescript
CV = 标准差 / 平均值

• CV < 0.3: 波动较小
• 0.3 ≤ CV < 0.5: 波动中等
• CV ≥ 0.5: 波动较大
```

**Z分数（Z-Score）**：
```typescript
Z = (X - μ) / σ

• |Z| > 2.5: 异常值（占1.24%）
• |Z| > 2.0: 可疑值（占4.55%）
• |Z| ≤ 2.0: 正常值（占95.45%）
```

### 2. 智能评分

**置信度计算**：
```typescript
let confidence = 100;

// 多维度扣分
confidence -= dataQuantityPenalty;
confidence -= volatilityPenalty;
confidence -= completenessPenalty;

// 保底机制
confidence = Math.max(confidence, 30);
```

### 3. 图表优化

**三Y轴配置**：
```typescript
scales: {
  y: { position: 'left' },    // 销售额
  y1: { position: 'right' },  // 人数
  y2: { position: 'right' }   // 效率
}
```
