# 📝 更新日志 V6.0

### 🎨 新增全局样式系统

#### CSS 变量定义
```css
:root {
  /* 背景色 */
  --bg-primary: #f8f9fa;
  --bg-secondary: #ffffff;

  /* 功能色 */
  --color-primary: #4a90e2;
  --color-success: #52c41a;
  --color-warning: #faad14;
  --color-danger: #f5222d;

  /* 阴影 */
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 6px rgba(0,0,0,0.1);
  --shadow-lg: 0 10px 15px rgba(0,0,0,0.1);

  /* 圆角 */
  --radius-sm: 6px;
  --radius-md: 8px;
  --radius-lg: 12px;
}
```

- `.modern-card` - 现代化卡片样式
- `.modern-button` - 现代化按钮样式
- `.modern-input` - 现代化输入框样式

### 📝 使用指南

#### 编辑班次
1. 进入 "排班室" 页面
2. 找到要编辑的班次卡片
3. 点击 "编辑" 按钮
4. 修改班次信息（名称/类型/时间/时长）
5. 点击 "更新班次" 保存

#### 编辑活动
1. 进入 "活动规划" 页面
2. 找到要编辑的活动卡片
3. 点击 "编辑" 按钮
4. 调整流量系数或修改描述
5. 点击 "更新活动" 保存

### ⚠️ 注意事项

1. **数据库自动升级**: 首次启动V6.0会自动添加 `update_time` 字段
2. **编辑权限**: 所有非默认方案都可以编辑
3. **删除限制**: 默认参数方案不可删除（其他都可以）
4. **数据验证**: 所有字段都有实时验证
