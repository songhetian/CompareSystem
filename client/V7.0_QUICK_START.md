## 📝 重构页面的标准流程

### 示例：重构任意页面

#### Step 1: 导入必要组件

```tsx
import { Card, Space, Button, Typography } from '@arco-design/web-react';
import { PageHeader, StatsCard } from '../components/common';
import { IconPlus } from '@arco-design/web-react/icon';

const { Text } = Typography;
```

#### Step 2: 使用页面容器

```tsx
export const YourPage = () => {
  return (
    <div className='page-container'>
      {/* 页面内容 */}
    </div>
  );
};
```

#### Step 3: 添加页面头部

```tsx
<PageHeader
  title='页面标题'
  subtitle='页面描述信息'
  icon={<IconPlus />}
  extra={
    <Space size={12}>
      <Button type='primary' icon={<IconPlus />}>
        新建
      </Button>
    </Space>
  }
/>
```

#### Step 4: 使用卡片布局

```tsx
<Card
  bordered={false}
  title='卡片标题'
  extra={<Button type='text'>更多</Button>}
  style={{
    marginBottom: 'var(--spacing-medium)',
  }}
>
  {/* 卡片内容 */}
</Card>
```

#### Step 5: 使用统计卡片

```tsx
<div style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: 16 }}>
  <StatsCard
    title='总销售额'
    value={12580}
    suffix='万'
    precision={2}
    trend='up'
    trendValue='+15.3%'
    color='#165DFF'
  />
</div>
```

## 🎨 常用组件示例

### Button（按钮）

```tsx
// 主要按钮
<Button type='primary' size='large' icon={<IconPlus />}>
  新建项目
</Button>

// 成功按钮
<Button type='primary' status='success' size='large'>
  保存
</Button>

// 危险按钮
<Button type='primary' status='danger' size='large'>
  删除
</Button>

// 次要按钮
<Button type='secondary' size='large'>
  取消
</Button>

// 文本按钮
<Button type='text'>
  查看详情
</Button>
```

### Input（输入框）

```tsx
// 基础输入
<Input
  placeholder='请输入内容'
  allowClear
  size='large'
/>

// 带前缀
<Input
  placeholder='搜索'
  prefix={<IconSearch />}
  allowClear
/>

// 数字输入
<InputNumber
  placeholder='请输入数字'
  min={0}
  max={100}
  step={1}
  precision={2}
/>

// 多行输入
<Input.TextArea
  placeholder='请输入内容'
  rows={4}
  maxLength={200}
  showWordLimit
/>
```

### Select（下拉选择）

```tsx
<Select
  placeholder='请选择'
  showSearch
  allowClear
  size='large'
  filterOption={(inputValue, option) =>
    option.props.children.toLowerCase().includes(inputValue.toLowerCase())
  }
>
  <Select.Option value='1'>选项1</Select.Option>
  <Select.Option value='2'>选项2</Select.Option>
  <Select.Option value='3'>选项3</Select.Option>
</Select>
```

### DatePicker（日期选择）

```tsx
// 单个日期
<DatePicker
  placeholder='请选择日期'
  format='YYYY-MM-DD'
  allowClear
  size='large'
/>

// 日期范围
<DatePicker.RangePicker
  placeholder={['开始日期', '结束日期']}
  format='YYYY-MM-DD'
  allowClear
  size='large'
/>
```

### Table（表格）

```tsx
<Table
  columns={columns}
  data={data}
  stripe
  hover
  borderCell
  pagination={{
    pageSize: 20,
    showTotal: true,
    showJumper: true,
    sizeCanChange: true,
    pageSizeOptions: [10, 20, 50, 100],
  }}
  scroll={{ x: 1200, y: 600 }}
/>
```

### Modal（模态框）

```tsx
<Modal
  title='标题'
  visible={visible}
  onOk={handleOk}
  onCancel={handleCancel}
  style={{ width: 600 }}
  okText='确定'
  cancelText='取消'
>
  <Form layout='vertical'>
    {/* 表单内容 */}
  </Form>
</Modal>
```

### Form（表单）

```tsx
<Form
  form={form}
  layout='vertical'
  size='large'
  autoComplete='off'
  onSubmit={handleSubmit}
>
  <Form.Item
    label='项目名称'
    field='name'
    rules={[
      { required: true, message: '请输入项目名称' },
      { minLength: 2, message: '至少2个字符' },
    ]}
  >
    <Input placeholder='请输入项目名称' />
  </Form.Item>

  <Form.Item>
    <Space>
      <Button type='primary' htmlType='submit'>
        提交
      </Button>
      <Button onClick={() => form.resetFields()}>
        重置
      </Button>
    </Space>
  </Form.Item>
</Form>
```

## 🐛 常见问题

### Q1: 样式不生效？
**A**: 确保已导入 `global.css`
```tsx
// main.tsx
import './theme/global.css';
```

### Q2: 找不到组件？
**A**: 检查导入路径
```tsx
import { PageHeader } from '../components/common';
```

### Q3: 设计令牌变量不识别？
**A**: 确保在 CSS 变量中使用
```tsx
style={{ padding: 'var(--spacing-medium)' }}
```

### Q4: Arco 组件样式覆盖？
**A**: 使用 `!important` 或提高选择器优先级
```css
.custom-button {
  background: #165DFF !important;
}
```
