## 🎨 设计系统成果

### 设计令牌 (Design Tokens)

**文件**: `src/theme/design-tokens.ts`

```typescript
- 颜色系统: 主色、功能色、中性色
- 间距系统: 8px 栅格 (mini → xxlarge)
- 圆角系统: small → circle
- 阴影系统: light → strong
- 字体系统: 大小、粗细、行高
- 过渡动画: fast, normal, slow
- 层级系统: base → tooltip
```

**文件**: `src/theme/global.css`

```css
- .page-container    - 页面容器

- .timeline-card     - 时间线卡片
```

## 🎓 最佳实践总结

### 1. 组件使用规范

```tsx
// ✅ 推荐：使用 PageHeader
<PageHeader title="标题" subtitle="描述" icon="🎯" />

// ❌ 不推荐：自定义头部
<div className="custom-header">...</div>
```

```tsx
// ✅ 推荐：使用 StatsCard
<StatsCard title="销售额" value={1000} suffix="万" icon="💰" />

// ❌ 不推荐：使用 Statistic
<Statistic title="销售额" value={1000} />
```

### 2. 样式使用规范

```tsx
// ✅ 推荐：使用 CSS 类
<div className="page-container">
  <div className="page-content">
    <div className="data-card">...</div>
  </div>
</div>

// ❌ 不推荐：内联样式
<div style={{ padding: 24, background: '#fff' }}>...</div>
```

```tsx
// ✅ 推荐：使用设计令牌
<div style={{
  padding: 'var(--spacing-large)',
  background: 'var(--gray-1)'
}}>

// ❌ 不推荐：硬编码值
<div style={{ padding: '24px', background: '#f7f8fa' }}>
```

### 3. 布局使用规范

```tsx
// ✅ 推荐：统一布局结构
<div className="page-container">
  <PageHeader {...props} />
  <div className="page-content">
    {/* 页面内容 */}
  </div>
</div>

// ❌ 不推荐：自定义布局
<div className="h-full flex flex-col">
  <div className="header">...</div>
  <div className="content">...</div>
</div>
```
