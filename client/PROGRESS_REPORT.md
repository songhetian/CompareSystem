## 🎨 技术亮点

### 1. 统计学算法应用

**变异系数（CV）**：
```typescript
CV = σ / μ
用途: 评估数据波动性
应用: 置信度评分
```

**Z分数（Z-Score）**：
```typescript
Z = (X - μ) / σ
用途: 检测异常值
应用: 异常数据标记
```

### 2. 数据校验系统

**六大维度**：
- 日期校验（格式+逻辑）
- 销售额校验（范围+异常）
- 人数校验（完整性+异常）
- 咨询量校验（基础）
- 转化率校验（范围+格式）
- 人均效率校验（逻辑）

**三级反馈**：
